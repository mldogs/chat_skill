Отправить сообщение в Telegram-чат проекта.

## Порядок действий

1. Составь сообщение по запросу пользователя. Правила:
   - Без markdown-разметки (plain text для Telegram)
   - Без длинных тире (используй обычные дефисы)
   - Лаконично и по делу

2. Покажи составленное сообщение пользователю через AskUserQuestion и дождись подтверждения.

3. Если одобрено, отправь:
```bash
python -m chat send "текст сообщения"
```

4. Подтверди отправку с ID сообщения.

ВАЖНО: Никогда не отправляй без подтверждения пользователя.
